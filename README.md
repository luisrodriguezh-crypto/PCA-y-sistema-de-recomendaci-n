# PCA-y-sistema-de-recomendaci-n
En este proyecto exploramos dos enfoques para trabajar con datos incompletos. Primero, utilizamos el dataset Wine de scikit-learn para comparar estrategias de imputación — media, KNN y Matrix Completion mediante SoftImpute — evaluando su impacto en un modelo de clasificación de regresión logística. Después, aplicamos el mismo algoritmo de completación de matrices al dataset MovieLens 100k para construir un sistema de recomendación colaborativa, donde la tarea central fue predecir los ratings faltantes en una matriz de 943 usuarios × 1,682 películas con solo el 6.3% de entradas conocidas. Para ello implementamos SoftImpute desde cero — basado en SVD iterativo con umbral suave sobre valores singulares — y generamos recomendaciones personalizadas ordenando los ratings estimados para películas no vistas por cada usuario.
